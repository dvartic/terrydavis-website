What is GodSays/GodSpeak

TempleOS includes a few random generators. Terry believed that these generators allowed God to speak and communicate with the user. Terry was seen on videos and on forums interpreting the output of these random generators, making associations in the text much like an Rorschach ink-blot test is performed.

From my dive into TempleOS, these are the random generators available (written in HolyC):

  • GodWord: God says a single word, taken from a list of 7570 common english words.
  • GodBiblePassage: God selects a random passage from the Bible for you to read.
  • GodSong: God makes a song for you.
  • GodDoodle: God makes a sprite for you.

According to Terry, the technique used to source randomness on those programs is reading a microsecond-range stop-watch each button press. You can read more about them on this Christine Dodrill post.

Apart from the TempleOS generators, Terry had a bash script on GNU+Linux that allowed God to construct more complex phrases. This is what we call GodSpeak/GodSays. This code used /dev/urandom as source of randomness, gshuff to shuffle the elements and a Happy.txt file that contained those elements. This Happy.txt file contained 712 entries of both words and phrases. The bash script outputs 32 of these elements.

I ported this bash script to JavaScript while trying to stay as faithful as possible to the original code. Check my GitHub page for more information.
